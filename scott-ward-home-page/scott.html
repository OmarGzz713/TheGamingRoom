About Me

I began a new chapter in my life after being laid off from my job as an ecommerce manager during the pandemic in 2020. Instead of seeking another job, I decided to continue my education as I was always working with technology but felt I never fully understood it. So I enrolled at the University of Houston Downtown in Spring of 2021 and declared Computer Science as my major.

Since starting back in school, I've had a child, excelled in my studies, and found a love for math, coding, learning, and game development.

I am on trek to graduate summa cum laude in spring of 2025 with a Bachelors of Science in Computer Science while minoring in Mathematics.

Projects

Don't Look Back

A slenderman inspired horror game created in Summer of 2024 for CS 3310 - Fundamentals of game development taught by Dr. Ling Xu.

Pokédex

A functioning Pokédex built using the Adafruit Memento camera. CircuitPython was used for programming the microcontroller while Python handled communication with ChatGPT for Pokémon identification. Made for a class project in CS 3319 - Internet of Things - taught by Dr. Ting Zhang.

UHgeekDom

A collborative project for CS 4300 - Web Development - taught by Dr. Ting Zhang. HTML, CSS, and Javascript were used to build out a functioning website.

Security Robot

A small-scale demo of a functioning security robot. The robot used was the Elegoo Smart Car Kit v4.0. It was programmed using the Arduino IDE for the onboard microcontroller, while the facial recognition was ran on PyCharm using a the wifi camera to connect to the car and provide real-time tracking using Haar cascades.

UHD Apartments

A semester-long project made for CS 3321 - Software Engineering. A simulated apartment management system built using C++, git for version control, and github for remote collaboration.

Hakiu-4-U

A program written for CS 3300 - Object-Oriented Programming with Java - as a class project. Takes user input and uses regular expressions to make sure that the input is correctly structured as a haiku (5-7-5 syllable structure). The program then outputs the haiku to a QR code.
